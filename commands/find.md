---
name: find
description: Find bugs.
---
Thoroughly analyze this repo and look for serious undiscovered bugs.

When you find one, implement a test in that demonstrates the bug.

Make sure that the test demonstrates the bug clearly is as simple as possible.

It must be an existing bug in the current codebase, not one that is already fixed.

Only demonstrate one bug, and prioritize the broadly applicable and serious.
For example, bugs that that cause incorrect results, corrupt data, or can lead
to a crash.
